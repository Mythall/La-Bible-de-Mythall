# Réflexes surhumains

Le personnage acquiert la capacité d'éviter de justesse les sorts à zone d'effet, comme les boules de feu que les mages peuvent lancer. Il réduit alors de moitié les dégâts qu'il aurait dû subir ou si le sort n'occasionne pas de dommages, il réduit la durée de moitié. Une pirouette est nécessaire pour éviter le sort à zone d'effet.

**Classes autorisées :** Toutes

**Classes défendues :** Aucune

**Races défendues :** Nain
