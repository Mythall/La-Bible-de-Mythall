# Désamorce de piège

Les pièges peuvent être désamorcés par les personnages détenant la fourberie *Désamorce de piège*.

Si le personnage a le niveau nécessaire pour détecter le piège, il peut ensuite essayer de le désamorcer. Cependant, il se peut qu'il n'ait pas le bon niveau pour le désamorcer.

## Choix du joueur

Ce qui veut dire que le joueur a deux possibilités :

- Ne pas ouvrir le piège et s'en aller
- Essayer de le désamorcer, mais le déclencher s'il n'a pas le niveau requis

S'il a au contraire le bon niveau de désamorce, il peut le désamorcer et récupérer les pièces du piège.

## Formule

**Niveau de désamorce de piège = (niveau de classe effectif de classe + dextérité) X 0.5**

(nivelé à la baisse)
