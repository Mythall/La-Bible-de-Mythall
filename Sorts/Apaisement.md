# Apaisement

**École** : Conjuration

## Caractéristiques

- **Niveau** : Prêtre 1, Chaman 1
- **Portée** : Personnelle
- **Durée** : Spéciale
- **Zone d'effet** : 5 Mètres de rayon
- **Incantation** : *Tlu valae da'palamur* + Symbole + cible

## Effet

Ce sort permet de redonner de l'assurance à tous les alliés présents dans la zone d'effet, conférant une résistance contre un effet de terreur. Si le sujet est déjà sous l'influence d'un sort ou effet de terreur, il sort immédiatement de sa torpeur. Le sort se dissipe après avoir combattu un effet de terreur par personne.

**Amélioration** : Ce sort compte comme une amélioration jusqu'à utilisation.
