# Barrière Élémentaire Mineure

**École** : Abjuration

## Caractéristiques

- **Niveau** : Ens/Mag 2, Prêtre 2, Druide 2, Paladin 2, Rôdeur 1
- **Portée** : Contact
- **Durée** : 1 Combat/10 minutes
- **Zone d'effet** : 1 Personne
- **Incantation** : *Obsulas qua'laen dronn* + Symbole + cible

## Effet

Ce sort donne à la cible un résistance de 1 contre les dégâts de feu, de froid **OU** de foudre pendant toute la durée du sort. Le lanceur de sort choisit le type de protection élémentaire du sort lorsqu'il le lance.

Ce sort ne se cumule pas avec lui-même, et ce même si l'élément affecté qui est choisi est différent.

**Amélioration** : Ce sort compte comme une amélioration.
