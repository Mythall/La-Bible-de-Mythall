# Bourrasque

**École** : Évocation

## Caractéristiques

- **Niveau** : Ens/Mag 3, Druide 2
- **Portée** : Porté de jet
- **Durée** : Instantanée
- **Zone d'effet** : Spécial
- **Incantation** : *Lil nida pholor dossta* + Symbole + cible

## Effet

Le lanceur de sort libère une grande énergie sous la forme d'un puissant vent. Toutes les personnes qui sont touchées par le riz doivent reculer de 3 mètres.
