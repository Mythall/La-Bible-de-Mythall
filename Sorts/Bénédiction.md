# Bénédiction

**École** : Enchantement

## Caractéristiques

- **Niveau** : Prêtre 1, Paladin 1, Guerre 1
- **Portée** : Personnelle
- **Durée** : 1 Combat/10 minutes
- **Zone d'effet** : 1 Personne par 2 NLS à 5 Mètres de rayon
- **Incantation** : *Usstan inbal xunor whol colbauth taga dos phuul bel'la del ussa sar lueth ussa quar'valuk.* + Symbole + cible

## Effet

Ce sort aide les personnes qui en bénéficient en leur donnant **1 PA** durant le prochain combat.

**Effet niv. 5** : Le sort confère aussi un bonus de **+1 point de dégât non cumulable** applicable sur les attaques physiques.

**Amélioration** : Ce sort compte comme une amélioration.
