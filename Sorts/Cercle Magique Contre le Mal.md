# Cercle Magique Contre le Mal

**École** : Abjuration

## Caractéristiques

- **Niveau** : Ens/Mag 3, Barde 3, Prêtre 3, Paladin 3, Bien 3, Mal 3
- **Portée** : Personnelle
- **Durée** : 1 Heure par 3 NLS
- **Zone d'effet** : 3 Mètres de rayon
- **Incantation** : *Ril l'abbilen taga usstan solar phuul kulggen del verin* + Symbole + cible

## Effet

Tous les alliés à 3 mètres de rayon du lanceur de sorts bénéficient des effets du sort Protection Contre le Mal. Ce sort constitue une barrière magique autour du sujet qui le protège des effets de charme. Le sort est dissipé après avoir bloqué un effet de ce type.

Pendant que le sort est actif, le bénéficiaire est immunisé aux attaques de ses propres invocations.

**Amélioration** : Ce sort compte comme une amélioration.
