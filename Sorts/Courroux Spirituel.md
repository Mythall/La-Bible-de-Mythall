# Courroux Spirituel

**Niveau** : Druide 6, Paladin 6, Bien 6, Destruction 5, Osayo 5, Justice 8

**Portée** : Personnelle

**Durée** : Instantanée

**Zone d'effet** : 5 Mètres de rayon

**Incantation** : *Taga lil nizzre' del ussa quar' valuk dossta zotreth* + Symbole + cible

**Effet** : Le jeteur de sorts canalise son énergie spirituelle pour la libérer sous forme d'éclairs dans toutes les directions. Toutes les créatures dans la zone d'effet (sauf le lanceur) sont vivement électrocutées et 1 point de dégât de foudre à toutes les parties du corps.
