# Cécité

**École** : Transmutation

## Caractéristiques

- **Niveau** : Ens/Mag 2, Barde 2, Prêtre 3
- **Portée** : 5 Mètres
- **Durée** : 5 Secondes par NLS
- **Zone d'effet** : 1 Personne
- **Incantation** : *Dossta naust sol* + Symbole + cible

## Effet

La cible subit un **effet de cécité** pour la durée.
