# Délivrance des Malédictions

**Niveau** : Ens/Mag 4, Barde 3, Prêtre 3, Chance 5, Chaman 4

**Portée** : Contact

**Durée** : Permanente

**Zone d'effet** : Spéciale

**Incantation** : *Usstan dossta valaer del verin'ol* + Symbole + cible

**Effet** : Ce sort permet de faire disparaître une seule malédiction affectant un individu ou encore de renvoyer une influence maléfique. Notez que le sort ne fait pas disparaître la malédiction d'un objet maudit, mais qu'il permet simplement de se débarrasser de l'objet. Certaines malédictions plus puissantes ne peuvent être contrées par ce sort ou ne peuvent l'être que si le personnage a atteint un certain niveau (prendre le NLS pour déterminer ce niveau).
