# Détection de l'Invisible

**Niveau** : Ens/Mag 4, Barde 6, Prêtre 4, Connaissance 4

**Portée** : Personnelle

**Durée** : 5 Minutes par NLS

**Zone d'effet** : Personnelle

**Incantation** : *Taga lil velkyn tlu ragar a ussa* + Symbole + cible

**Effet** : Le personnage voit normalement les objets et créatures invisibles, ainsi que ceux qui évoluent dans le plan Éthéré. Les créatures se trouvant dans le plan Éthéré prennent la forme de silhouettes translucides, ce qui permet au personnage de faire la différence entre les sujets visibles, invisibles et éthérés. Le sort ne dévoile pas la méthode employée pour se rendre invisible.

**Amélioration :** Ce sort compte comme une amélioration.
