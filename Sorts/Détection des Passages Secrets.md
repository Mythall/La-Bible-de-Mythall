# Détection des Passages Secrets

**Niveau** : Ens/Mag 1, Rôdeur 1, Connaissance 1

**Portée** : Personnelle

**Durée** : 10 Minutes pas NLS

**Zone d'effet** : 3 Mètres de rayon

**Incantation** : *Taga lil brornen tlu dedoer ssussun* + Symbole + cible

**Effet** : Le lanceur détecte tous les passages secrets dans la zone. Cela lui permet de lire les papiers hors-jeux identifiant les passages secrets (en Ombre-terre par exemple). Le sort peut également aider à avancer certaines quêtes (voir animation).

**Amélioration :** Ce sort compte comme une amélioration.
