# Feu Solaire

**Niveau** : Ens/Mag 5, Magie 4, Linvala 4, Druide 4

**Portée** : Personnelle

**Durée** : Instantanée

**Zone d'effet** : 5 Mètres de rayon

**Incantation** : *Streea a lil chath* + Symbole + (cri de « Kamikaze » en option) + cible

**Effet** : Le Feu Solaire est une explosion de flammes invisibles pour ceux qui ne voient pas la magie. Les flammes s'abattent sur les cibles dans la zone d'effet et leur inflige 1 points de dégâts de feu sur toutes les parties du corps.
