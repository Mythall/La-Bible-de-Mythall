# Forme d'Arbre

**Niveau** : Druide 4, Rôdeur 5, Vynash 2

**Portée** : Personnelle

**Durée** : Spéciale

**Zone d'effet** : Personnelle

**Incantation** : *Usstan uil lil larnae* + Symbole + cible

**Effet** : Grâce à ce sort, le personnage peut prendre la forme d'un petit arbre vivant, d'un buisson, d'un tronc d'arbre mort, etc. Même en inspectant de près, il est impossible de déterminer que l'arbre est en réalité une créature vivante. Tout montre que le personnage est un arbre. Le personnage garde toutes ses facultés et ses sens, mais ne peut ni bouger, ni lancer de sorts, ni parler. Il peut revenir sous sa forme humanoïde quand il le désire. Lorsqu'une personne est sous l'effet de Forme d'arbre, elle doit faire un symbole d'hors-jeu. Si le personnage lance le sort alors que quelqu'un l'observe, la personne qui voit le sort en action sait quel arbre le lanceur de sorts est devenu. Il n'est pas possible d'utiliser les sorts de détection conventionnels pour repérer le lanceur de sorts.

**Amélioration :** Ce sort **ne** compte **pas** comme une amélioration.
