# Glissement

**Niveau** : Druide 2

**Portée** : 5 mètres

**Durée** : Instantanée

**Zone d'effet** : 1 Personne par 3 NLS

**Incantation** : *Lil har'dro zhah ussa abban, shal nausbyr dossa dindwel* + Symbole + cible

**Effet** : Les personnes affectées par le sort doivent tomber au sol. Il est obligatoire que le ventre ou le dos touche au sol avant de se relever.
