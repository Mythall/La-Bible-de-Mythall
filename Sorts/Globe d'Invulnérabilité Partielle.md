# Globe d'Invulnérabilité Partielle

**Niveau** : Ens/Mag 4, Barde 4, Protection 4

**Portée** : Personnelle

**Durée** : 1 Combat/10 minutes

**Zone d'effet** : Personnelle

**Incantation** : *Elend faerin kulggen* + Symbole + cible

**Effet** : Confère une immunité contre le prochain sort offensif le ciblant, peu importe lequel. Seul Dissipation Suprême peut dissiper ce sort.

**Amélioration :** Ce sort compte comme une amélioration.
