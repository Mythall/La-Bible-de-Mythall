# Guérison Multiple

**Niveau** : Prêtre 5, Druide 6, Barde 5, Guérison 5, Chaman 5

**Portée** : Contact

**Durée** : Instantanée

**Zone d'effet** : 5 Mètres de rayon

**Incantation** : *A ussa sar lueth ussa quar'valuk, ril l'abbilen usstan solar tlu bwael nelgetha* + Symbole + cible

**Effet** : Ce sort rend 1 PV à toutes les parties du corps de toute créature amicale située dans un rayon de 5 m.
