# Guérison Suprême de Groupe

**Niveau** : Prêtre 8, Druide 9, Guérison 9

**Portée** : Contact

**Durée** : Instantanée

**Zone d'effet** : Tous les alliés compris dans un rayon de 5 m

**Incantation** : *Lil mzilst malla nelgetha del jal lil quar'valuk ul'naus whol lil dro akifa* + Symbole + cible

**Effet** : Ce sort guérit instantanément toutes les blessures des sujets. Le sujet retrouve tous ses points de vie. Ce sort n'a aucun effet sur les morts-vivants.
