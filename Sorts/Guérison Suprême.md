# Guérison Suprême

**Niveau** : Prêtre 6, Druide 7, Guérison 6

**Portée** : Contact

**Durée** : Instantanée

**Zone d'effet** : 1 Personne

**Incantation** : *L'Elamshinae lueth nelgetha ortahe a ussa quar'valuk* + Symbole + cible

**Effet** : Ce sort guérit instantanément toutes les blessures du sujet. Le sujet retrouve tous ses points de vie. Ce sort n'a aucun effet sur les morts-vivants.
