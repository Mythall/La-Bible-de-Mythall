# Image Miroir Mineure

**Niveau** : Ens/Mag 1, Barde 1, Prêtre 1, Duperie 1, Justice 1

**Portée** : Personnelle

**Durée** : Spéciale

**Zone d'effet** : Personnelle

**Incantation** : *Draa'l del ussa* + Symbole + cible

**Effet** : Ce sort permet de créer une image du jeteur de sorts qui reste à ses côtés. Cette image correspond à 1 PA. Le sort perdure jusqu'à utilisation.

**Amélioration :** Ce sort compte comme une amélioration.
