# Mot de Pouvoir Aveuglant

**Niveau** : Ens/Mag 8, Guerre 7

**Portée** : 5 Mètres

**Durée** : Permanent

**Zone d'effet** : 1 Personne

**Incantation** : *Na'sol* + cible

**Effet** : Ce sort aveugle la cible, elle doit fermer les yeux. Au cas où la personne doive se déplacer seule, elle peut ouvrir les yeux suffisamment pour voir où elle va pour sa sécurité.
