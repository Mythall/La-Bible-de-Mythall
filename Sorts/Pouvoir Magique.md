# Pouvoir Magique

**Niveau** : Magie 0

**Portée** : Personnelle

**Durée** : 1 Événement

**Zone d'effet** : Personnelle

**Incantation** : *Usstan belbaue lil faer wund nindol* + Symbole + cible

**Effet** : Ce sort permet de placé un pouvoir magique dans l'objet du choix du lanceur de sorts. Le lanceur choisi un sort et le place dans l'objet, il peut alors le lancer en utilisant l'incantation. L'objet contient 1 charge plus 1 par 3 NLS. Les sorts placés dans des objets sont inutilisable durant un événement. Lancer un sort avec l'objet ainsi créé ne coûte plus de points de mana, mais utilise des charges. Peut être utilisé par d'autres personnages. Peut créer 1 item par 2 NLS. Chaque item doit utiliser un sort différent. Seul les sorts de prêtre peuvent être utilisés pour ce sort. Les objets ainsi créés ne perdurent pas entre les événements.
