# Protection Anti-tempêtes

**Niveau** : Druide 3, Rôdeur 3, Chance 3, Protection 3

**Portée** : Personnelle

**Durée** : 1 Combat/10 minutes

**Zone d'effet** : Personnelle

**Incantation** : *Usstan inbal uss ky'ov'aer qua'laen lil llar dronn* + Symbole + cible

**Effet** : Ce sort confère une résistance de 2 dégâts contre le feu, le froid et la foudre.

**Amélioration** : Ce sort compte comme une amélioration.
