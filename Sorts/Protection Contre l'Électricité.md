# Protection Contre l'Électricité

**Niveau** : Ens/Mag 5, Prêtre 5, Druide 5, Rôdeur 4

**Portée** : Contact

**Durée** : 1 Combat/10 minutes

**Zone d'effet** : 1 Personne

**Incantation** : *Ky'ov'aer qua'laen nizzre'* + Symbole + Cible

**Effet** : Ce sort confère une résistance de 3 dégâts contre la foudre.

**Amélioration** : Ce sort compte comme une amélioration.
