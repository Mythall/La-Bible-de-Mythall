# Protection Contre le Feu

**Niveau** : Ens/Mag 3, Prêtre 3, Druide 3, Rôdeur 2

**Portée** : Contact

**Durée** : 1 Combat/10 minutes

**Zone d'effet** : 1 Personne

**Incantation** : *Ky'ov'aer qua'laen chath* + Symbole + cible

**Effet** : Ce sort confère une résistance de 3 dégâts contre le feu.

**Amélioration** : Ce sort compte comme une amélioration.
