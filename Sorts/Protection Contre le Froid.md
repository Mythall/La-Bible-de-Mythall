# Protection Contre le Froid

**Niveau** : Ens/Mag 4, Prêtre 4, Druide 4, Rôdeur 3

**Portée** : Contact

**Durée** : 1 Combat/10 minutes

**Zone d'effet** : 1 Personne

**Incantation** : *Ky'ov'aer qua'laen inthuul* + Symbole + cible

**Effet** : Ce sort confère une résistance de 3 dégâts contre le froid.

**Amélioration** : Ce sort compte comme une amélioration.
