# Protection Contre le Mal

**Niveau** : Prêtre 1, Paladin 1, Barde 1, Bien 1, Mal 1

**Portée** : Contact

**Durée** : 1 Heure par NLS

**Zone d'effet** : 1 Personne

**Incantation** : *Kulggen del verin* + Symbole + cible

**Effet** : Ce sort constitue une barrière magique autour du sujet, ce qui le protège contre les effets de type charme. Le sort est dissipé après une attaque de ce type. Pendant que le sort est actif, le bénéficiaire est immunisé aux attaques de ses propres invocations.

**Amélioration** : Ce sort compte comme une amélioration.
