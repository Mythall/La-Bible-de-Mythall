# Protection Contre les Projectiles

**Niveau** : Ens/Mag 2

**Portée** : Contact

**Durée** : 3 Heures

**Zone d'effet** : 1 Personne

**Incantation** : *Ky'ov'aer qua'laen lil b'luth'ol* + Symbole +cible

**Effet** : Cette protection crée une barrière magique entourant la cible du sort. Cette barrière annule toutes attaques contre tous les projectiles non-magiques. Cela peut être une flèche, un carreau, une dague de jet, etc. Cette protection dure 3 heures.

**Amélioration** : Ce sort compte comme une amélioration.
