# Rage de Malavon

**Niveau** : Ens/Mag 7

**Portée** : Personnelle

**Durée** : Instantanée

**Zone d'effet** : 5 Mètres de rayon

**Incantation** : *Malavon jiv'undus* + Symbole + cible

**Effet** : Le corps du magicien projette dans toutes les directions des rayons d'énergie noire, néfaste pour ses alliés comme pour ses ennemis. Les barbillons magiques infligent 1 point de dégâts magiques à toutes les parties du corps des personnes qu'ils touchent. Le sort touche tout le monde dans la zone d'effet, amis comme ennemis.
