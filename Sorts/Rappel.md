# Rappel

**Niveau** : Ens/Mag 6, Druide 6, Connaissance 6

**Portée** : Contact

**Durée** : 15 Minutes

**Zone d'effet** : Personnelle

**Incantation** : *Ghil, pholor nindol har, usstan orn doer* + Symbole + cible

**Effet** : Grâce à ce sort, le lanceur de sort, peut enchanter le sol pour en faire un point de Rappel. Le lanceur de sort se téléportera à cet endroit bien précis s'il relance le sort (incantation et gestuelle). Il est impossible pour une personne autre que le lanceur de sort d'identifier le sol enchanté, donc aucun moyen de dissiper le sort. Le sort se dissipe de lui-même après 15 minutes. Le lanceur de sorts doit réciter l'incantation 2 fois en tout : une fois pour créer le point de rappel et une fois pour y revenir. Les points de mana sont utilisés pour enchanter le sol, mais reste gratuit pour le retour.

**Amélioration** : Ce sort **ne** compte **pas** comme une amélioration.
