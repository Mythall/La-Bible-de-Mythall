# Vapeur Colorée

**Niveau** : Ens/Mag 1, Barde 1, Linvala 1

**Portée** : Jet de riz

**Durée** : 5 Secondes

**Zone d'effet** : Jet de Riz

**Incantation** : *Tlu na'sol* + Symbole + Jet de riz

**Effet** : Ce sort fait paraître le riz dans la main du magicien comme un arc-en-ciel de couleurs, qu'il peut lancer sur les personnes de son choix. Les personnes touchées sont aveuglées pendant 5 secondes.
