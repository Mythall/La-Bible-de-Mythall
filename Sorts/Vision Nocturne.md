# Vision Nocturne

**Niveau** : Ens/Mag 3, Rôdeur 3, Barde 3, Druide 3

**Portée** : Contact

**Durée** : 1 Heure par NLS

**Zone d'Effet** : 1 Personne

**Incantation** : *Usstan sol wund l'oloth* + Symbole + cible

**Effet** : Le bénéficiaire acquiert la faculté de voir dans l'obscurité la plus complète. Les formes sont bien définies, mais il voit en noir et blanc. Ce sort est particulièrement utile pour les aventuriers qui désirent se promener en ombre-terre.

**Amélioration** : Ce sort compte comme une amélioration.
