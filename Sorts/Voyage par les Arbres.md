# Voyage par les Arbres

**Niveau** : Druide 5

**Portée** : Spéciale

**Durée** : 1 Événement

**Zone d'Effet** : Spéciale

**Incantation** : *Taga lil larnae oplynnas a ussa uss colbauth sreen'aur whol z'hind* + Symbole + cible

**Effet** : Le personnage enchante deux arbres par lesquels il pourra par la suite voyager. Une fois les arbres enchantés, il n'y a aucun signe distinctif ou moyen quelconque de déterminer que les arbres sont affectés par ce sort. Seul le personnage ayant lancé le sort peut voyager ainsi. Le voyage se fait avec le symbole "hors jeu" et sans détours. Si le personnage relance le sort, c'est pour modifier l'un des arbres de destination.
