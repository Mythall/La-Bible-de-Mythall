# Émotion Terreur

**Niveau** : Ens/Mag 4, Barde 4

**Portée** : Personnelle

**Durée** : 20 secondes

**Zone d'effet** : 3 Mètres de rayon

**Incantation** : *Lil alurl palamur plynn dossta* + Symbole + cible

**Effet** : Ce sort génère une violente terreur dans la zone indiquée. Toutes les créatures affectées subissent un effet de terreur pendant 20 secondes.
